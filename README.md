## 🚀 Quick Start

1. **Open the website:**
   ```bash
   open portfolio-website/index.html
   ```

2. **Or serve locally:**
   ```bash
   cd portfolio-website
   python3 -m http.server 8000
   # Visit http://localhost:8000
   ```
